The World wide web is a network that connects servers, swtiches and routers, we will focus this course on one Language that is used to create/display the information from the servers HTML

What is HTML

Hypertext Markup Language is a programing language that works with hypertext to move between pages, it use Markup or Tags to change the propertys of the text

Tags

In the following section we will review important tags and explain them:

  2. Italic = The italict tag, easy but not intuite Italic
  3. Links <a href="url" text </a> example udacity
